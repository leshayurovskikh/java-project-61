# **Игры разума**

---

- Проект представляет собой интерактивное действие с пользователем.

- Пользователю предоставляется меню с выбором игры.

- Запуск игры происходит по вводу значения игры в меню.

--------

**Проект реализует следующие игры:**

1. "Приветствие" ()
